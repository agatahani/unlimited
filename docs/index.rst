Cara Instal APK Android dari Luar App Store
###########################################

Cara install Apk Android dari sumber tidak dikenal sering dianggap sebagai langkah ilegal yang akan membahayakan perangkat kita. Benarkah demikian? Sebenarnya tidak selalu benar, karena pada praktiknya kita tetap bisa install Apk sumber tak dikenal dengan aman.

Artikel ini akan menjelaskan tentang bagaimana cara install aplikasi Android namun berasal dari sumber yang tidak dikenal, artinya aplikasi tersebut tidak diinstal langsung dari `Google Play Store`_. Di dalamnya juga akan memuat tentang hal-hal apa saja yang harus diperhatikan saat install aplikasi dari sumber yang tidak dikenal.

Hal yang Harus Diperhatikan Saat Install Aplikasi Dari Sumber yang Tidak Dikenal
-----------------------

Sebelum mengunduh aplikasi apapun dari laman yang tidak dikenal, sebaiknya kamu perlu perhatikan beberapa hal berikut ini:

1. Berasal dari Sumber Terpercaya
------------------

Pastikan bahwa aplikasi yang kamu unduh memang berasal dari sumber terpercaya. Hal ini bertujuan untuk meminimalisir kemungkinan masuknya virus ke smartphone hingga mengakibatkan kerusakan.

2. Hindari Install Aplikasi Modifikasi
------------------

Selain dari situs resmi, kamu juga perlu menghindari aplikasi yang sifatnya modifikasi. Aplikasi semacam ini umumnya bukan dari situs resmi tetapi dari website lain seperti mediafire, ShareIt, dll.

Aplikasi seperti ini sangat rentan dengan masuknya virus yang nantinya bisa merusak smartphone. Umumnya aplikasi ini selalu ditandai dengan kata-kata Mod Apk di belakang nama aplikasi. 

Cara Instal Apk Android Dari Sumber Tidak Dikenal
Setelah proses download selesai dilakukan, kamu bisa menginstal aplikasi tersebut di smartphone. Untuk proses instalasinya dapat dilakukan dengan dua cara dan berikut ulasannya:

#1. Install Manual dari Smartphone
------------------

Untuk proses instalasi aplikasi dapat dilakukan secara manual dengan cara sebagai berikut:

- Langkah 1: Kamu bisa membuka menu pengaturan pada smartphone.
- Langkah 2: Dari sekian menu yang ada, kamu bisa memilih privasi dan keamanan. 
- Langkah 3: Pada bagian tengah, kamu bisa mencentang izinkan install dari sumber tidak dikenal. 
- Langkah 4: Kamu bisa membuka file manager lalu cari aplikasi yang telah di download. 
- Langkah 5: Klik aplikasi lalu pilih install. 
- Langkah 6: Kamu bisa mengikuti berbagai petunjuk yang ada hingga aplikasi terinstal di smartphone. 

Rata-rata proses instalasi tersebut dapat dilakukan dengan cara di atas meskipun ada beberapa tipe lain yang berbeda. Misalnya Xiaomi baik di MIUI terbaru atau yang lawas.

Untuk perbedaannya tidak begitu signifikan dan hanya nama-nama menunya yang berbeda. Misalnya untuk MIUI 12 dimana kata-kata privasi dan keamanan diganti menjadi password and security. 

Selain kata-kata itu selebihnya sama dimana pengguna juga perlu mencentang atau memilih izinkan install dari sumber tidak dikenal. Sedangkan untuk di MIUI yang lawas juga demikian perbedaannya hanya di nama menunya. 

Jika di MiUI lawas, setelah pengguna memilih menu pengaturan lalu pilih **additional setting**. Selebihnya proses instalasi sama termasuk langsung menginstal aplikasi di smartphone. 

#2. Install Dari Google Chrome
------------------

Selain dari proses install manual di smartphone, kamu juga bisa melakukan dari browser Google Chrome. Untuk proses di instal, kamu perlu memastikan browser di update ke versi terbaru lalu ikuti petunjuk di bawah ini:

- Langkah 1: Download terlebih dahulu aplikasi dengan browser Google dan pastikan dari sumber yang dikenal.
- Langkah 2: Di hasil download browser, kamu bisa mengklik open pada aplikasi yang di unduh.
- Langkah 3: Ketika di klik akan muncul instruksi dan kamu bisa memilih setting yang ada di bagian kanan. 
- Langkah 4: Pada halaman selanjutnya ini, kamu tinggal memilih slider yang ada di sisi kanan menu. 
- Langkah 5: Terakhir, kamu bisa kembali ke menu sebelumnya lalu klik install. 

Untuk melakukan proses instalasi dengan cara ini, kamu harus memastikan Google Chrome sudah di update. Sebab, dari berbagai pengalaman kegagalan melakukan proses instalasi ini karena browser tidak di update.

Jadi itulah dua  cara instal apk dari sumber tidak dikenal baik dari smartphone atau browser Google Chrome. Saat instalasi, kamu perlu memastikan aplikasi tersebut berasal dari sumber yang terpercaya.
